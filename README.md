# Student Performance & Habit Tracker (Python)

This is a Python-based command-line application that allows users to log daily habits and analyze weekly performance trends.

# Features
- Logs sleep, study time, workout minutes, and stress level
- Supports multi-day data entry
- Calculates weekly averages
- Provides performance-based feedback
- Automatically saves a weekly report to a file

# Technologies Used
- Python 3
- Lists & loops for data processing
- File I/O for report persistence

# Why This Project
This project was built to apply foundational computer science skills to real-world performance tracking and discipline optimization.
